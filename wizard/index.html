Set Up Your Database

Step 1: Create a Supabase project

  1. Go to supabase.com and sign up (free).
  2. Click New Project, pick a name and region, wait for it to be ready.

Step 2: Generate an Access Token

This is a one-time token used to create your database table. It is not stored anywhere.

Create Access Token →

Step 3: Enter your credentials

Find the URL and Anon Key in your project's Settings → API page.

Download Your WebGIS

Everything is ready. Download the complete app, push to GitHub, enable Pages — your site is live.

Deploy in 3 Steps

  1. Unzip the downloaded file.
  2. Create a GitHub repository and push the contents.
  3. Go to Settings → Pages, set source to main branch. Your site is live!
