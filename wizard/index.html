How to Deploy

  1. Create a Supabase project (free tier is fine).
  2. Go to SQL Editor and run setup.sql, then data.sql.
  3. Copy your project URL and anon key from Settings → API.
  4. Open webgis.config.json, paste the URL and key in the supabase section.
  5. Put all files from the app/ folder + webgis.config.json in a GitHub repository.
  6. Enable GitHub Pages (Settings → Pages → Source: main branch).
  7. Your WebGIS site is live!
